# Sawmill

Processes wood into planks.

## Function

| Input | Output | Ratio |
|-------|--------|-------|
| Wood | Planks | 2:1 |

## Interaction

1. Player approaches sawmill with wood in backpack
2. Wood automatically transfers to sawmill
3. Processing happens over time
4. Planks appear for pickup

## Unlocking

Found in hexes as construction sites. Requires resources to build.

## World Availability

Present in all worlds that require planks for upgrades.
