# Smelter / Refinery

Processes ore into ingots and gems into refined versions.

## Smelter Function

| Input | Output | World |
|-------|--------|-------|
| Iron Ore | Iron Ingots | All early worlds |
| Copper Ore | Copper Ingots | Asium |

## Refinery Function

| Input | Output |
|-------|--------|
| Amethyst | Refined Amethyst |
| Azurite | Refined Azurite |
| Ruby | Refined Ruby |
| Sulphur | Refined Sulphur |
| Uranium | Refined Uranium |

## Interaction

1. Player approaches facility with raw materials
2. Materials automatically transfer
3. Processing happens over time
4. Refined materials appear for pickup

## Unlocking

- Smelter: Early worlds, basic resource cost
- Refinery: Later worlds (Odysseum+), higher resource cost
