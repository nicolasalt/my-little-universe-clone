# Upgrade Station

Facility for upgrading tools and armor.

## Function

Allows player to spend resources to upgrade:
- Pickaxe
- Axe
- Sword
- Helmet
- Armor

## Upgrade UI

1. Player interacts with station
2. UI shows all equipment and their levels
3. Cost displayed for next upgrade
4. Confirm to spend resources and upgrade

## Requirements

### Materials by World

| World | Wood Material | Metal Material |
|-------|--------------|----------------|
| Gaia - Wadirum | Wood/Planks | Iron Ingots |
| Dragonora | Hardwood | Iron Ingots |
| Odysseum | Hardwood | Marble |
| Egyptium | Hardwood | Processed Sandstone |
| Asium | Hardwood | Copper Ingots |

### Additional Materials
Higher tiers also require:
- Gems (Ruby, Amethyst, etc.)
- Refined gems (later worlds)
- World-specific materials

## World Availability

Each world has at least one upgrade station, usually near the spawn area.
