# Input Configuration

Input action mappings for MLU-style controls.

## Actions

| Action | PC Binding | Mobile |
|--------|------------|--------|
| move | WASD | Virtual joystick / Swipe |
| sprint | Shift | - |
| attack | Left Click / Space | Tap |
| charge_ability | Hold attack | Hold |
| rotate_camera | Right Click / Q / E | Drag |
| change_tool | V | Auto / Button |
| passive | C | Button |
| interact | F | Auto |
| map | M | Button |
| inventory | I | Button |
| pause | Escape | Button |

## Configuration
Added to `project.godot` under `[input]`.

## Notes
- Attack/Gather is automatic when near resources
- Tool switching can be set to Manual in options
- Mobile uses touch controls with virtual joystick
