# Input Configuration

Input action mappings for player controls.

## Actions

| Action | Default Binding |
|--------|-----------------|
| move_forward | W |
| move_back | S |
| move_left | A |
| move_right | D |
| jump | Space |
| sprint | Shift |
| interact | E |
| pause | Escape |

## Configuration
Added to `project.godot` under `[input]`.
